# Footer

Renders a footer section with navigation links, newsletter signup, and animated logo.

## Purpose

This component displays a comprehensive footer with navigation links, social media links, newsletter subscription form, contact information, and an animated SVG logo. The layout is responsive and includes dark mode support.

## Demo

```tsx demo
import { Footer } from '@/components/blocks/spike/footer';

export function Demo() {
  return (
    <Footer />
  )
}
```
