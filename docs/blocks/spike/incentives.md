# Incentives

Renders an incentives section with a carousel of benefit cards.

## Purpose

This component displays a section highlighting company benefits or incentives with a title, description, call-to-action button, and a carousel of incentive cards. Each card includes an icon, title, and description. The layout is responsive with mobile carousel and desktop grid views.

## Demo

```tsx demo
import { Incentives } from '@/components/blocks/spike/incentives';

export function Demo() {
  return (
    <Incentives />
  )
}
```
