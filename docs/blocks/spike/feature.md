# Feature

Renders a feature showcase section with a carousel of feature cards.

## Purpose

This component displays a collection of features in a carousel format, each with an icon, title, label, description, and background image. It includes navigation controls and responsive design that adapts to different screen sizes.

## Demo

```tsx demo
import { Feature } from '@/components/blocks/spike/feature';

export function Demo() {
  return (
    <Feature />
  )
}
```
