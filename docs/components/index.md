# Components Overview

This library includes 55+ UI components from shadcn/ui, all pre-configured and ready to use.

## Component Categories

### Form Components
- Button, Input, Textarea, Select, Checkbox, Switch, Radio Group
- Form, Field, Label, Input Group, Input OTP
- Native Select, Combobox

### Layout Components
- Card, Separator, Aspect Ratio, Scroll Area
- Resizable, Sidebar, Sheet, Drawer

### Overlay Components
- Dialog, Alert Dialog, Popover, Tooltip
- Hover Card, Dropdown Menu, Context Menu
- Command, Sheet, Drawer

### Navigation Components
- Breadcrumb, Navigation Menu, Menubar
- Pagination, Tabs

### Feedback Components
- Alert, Toast (Sonner), Progress, Skeleton
- Empty, Spinner

### Data Display
- Table, Chart, Avatar, Badge
- Accordion, Collapsible, Carousel
- Calendar

### Interactive Components
- Toggle, Toggle Group, Button Group
- Slider, Switch, Item

## Usage

Each component can be imported from `@/components/ui/[component-name]`:

```tsx
import { ComponentName } from "@/components/ui/component-name"
```

Browse the sidebar to see detailed documentation for each component.
