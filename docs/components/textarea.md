# Textarea

Displays a form textarea or a component that looks like a textarea.

## Usage

### Basic Usage

```tsx demo
import { Textarea } from "@/components/ui/textarea"

export function TextareaDemo() {
  return <Textarea placeholder="Type your message here." />
}
```
